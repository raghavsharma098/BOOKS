# Book Website — Next.js + TypeScript + Tailwind

Starter responsive site scaffolded with Next.js (App Router), TypeScript and Tailwind CSS.

Getting started

1. Install dependencies
   npm install
2. Run development server
   npm run dev

Features included
- App Router (Next.js)
- Tailwind CSS with dark mode
- Responsive header + footer and homepage
- Simple client-side contact form demo

Deploy
- Deploy on Vercel for best compatibility. A `vercel.json` is included.
